# AutoClicker
  制作者自身のコード学習、個人使用を目的としてとして
  Javaで作成したWindows11用の簡易的なオートクリッカーです。
  Tabキーを押すことでクリッカーの開始、停止を制御でき、
  クリッカーが動作しているときは自動で一定間隔でクリックを繰り返すことができます。

## 動作環境
  以下の環境において正常に動作することを確認しています。
  ・Java-21
  ・Windows11
  ・JNativeHook 2.2.2

### macOSでの動作について
  macOS環境でも本ツールは正常に動作することを確認しておりますが、
  マウス入力について、右クリックと左クリックが逆になる挙動を確認しています。
  本ツールはWindows11で使用することを想定しておりますので補正処理は行っていません。

## 使用方法
  本ツールを起動すると簡易的な設定画面が表示されます。
  以下は設定項目です。
  ・Interval オートクリックの間隔を整数で指定することができます。
    初期設定の100の場合ではおおよそ1秒間に10回クリックすることが可能です。
  ・Toggle/Momentary オートクリックの制御方法を変更できます。
    Toggleでは1回Tabキーを押すとクリック開始、もう一度押すとクリック終了となります。
    MomentaryではTabキーを押している間だけクリックを行います。
  ・マウスの画像上にある二つのボタンについて、
    それぞれクリックを行っている時、左クリックを行うか、右クリックを行うかを設定できます。
    クリックを行っている間このボタンがオンになっているクリック操作のみを行います。
    初期状態の両方オフの場合には動作中にもクリック操作が行われません。
  以上の設定を行ったうえでTabキーを押すことでクリックが開始します。
  
## 使用ライブラリ
  JaNativeHook 2.2.2
  キーボード入力を取得するために使用しています。
